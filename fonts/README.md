# Fonts

The grab bag: the curated SIL-OFL faces behind Marquee's closed `font` name list. The
vocabulary (renderer + `css/marquee.css` fallback stacks) already exists; this directory holds
the actual WOFF2 files and, alongside each, its OFL license text — the OFL permits exactly
this redistribution, and requires the license ride along.

**Self-hosted on principle:** an embedder serves these files itself. Linking a third-party
font CDN leaks every reader's IP per page view — fonts are a tracking vector, and care-modes
apply to them like any other fetch.

Layout (once populated):

```
fonts/
  orbitron.woff2
  orbitron.OFL.txt
  vt323.woff2
  vt323.OFL.txt
  ...
```

File names match the vocabulary tokens (see the table in [WRITING.md](../WRITING.md)).
`css/fonts.css` (generated by `npm run fetch-fonts` in `ts/html_renderer`) carries the
`@font-face` rules; until a face loads, every name degrades to its fallback stack in
`marquee.css`. Latin subsets only — these are display faces, not body-text workhorses.

Status: **31 faces landed**, each with its license text beside it. Thirty are SIL OFL; the
one exception is `special-elite` (Astigmatic), which is **Apache 2.0** — equally
redistributable, license included as `special-elite.LICENSE.txt`.
